# IPScan
*Scanning IP addresses*
___

> #### Description 
> Scanning a range of ip addresses with\without a port.

> #### Usage 
> * `ipscan -ip 192.168.0.77`
> * `ipscan -ip 192.168.0.77 -p 80`
> * `ipscan -ip 192.168.0.1-192.168.88.255 -p 80`
